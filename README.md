# Ez-Frontend
Frontend Angular 21 + Backend Spring Boot
## 🚀 Inicio rápido
\\\ash
cd ez-frontend
npm install
npm start
\\\
Abre http://localhost:4200
## 📁 Estructura
- **src/app/components/** - Componentes por rol (admin, ingeniero, usuario)
- **src/app/services/** - Servicios HTTP
- **src/app/app.routes.ts** - Rutas configuradas
## 📚 Stack
- Angular 21.1.0
- TypeScript 5.9.2
- RxJS 7.8.0
## 🔌 Backend
- Puerto: 8080
- Base URL: http://localhost:8080/api
## Scripts
- \
pm start\ - Servidor desarrollo
- \
pm run build\ - Build producción
- \
pm test\ - Tests
